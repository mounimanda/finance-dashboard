# 💰 Finwise - Finance Dashboard

A clean, interactive finance dashboard that helps users track transactions, visualize spending patterns, and manage personal finances. Built with vanilla JavaScript, Tailwind CSS, and Chart.js.

## 📋 Features Checklist

### ✅ Core Requirements Completed

| Requirement | Implementation |
|-------------|----------------|
| **Dashboard Overview** | 3 summary cards showing Total Balance, Income, and Expenses with real-time calculations |
| **Time-based Visualization** | Monthly trend bar chart comparing income vs expenses (Feb vs Mar) |
| **Categorical Visualization** | Pie chart showing spending breakdown by category (Food, Shopping, Utilities, etc.) |
| **Transactions Section** | Complete list with Date, Description, Category, Amount, and Type |
| **Filtering** | Filter by transaction type (All / Income / Expense) |
| **Sorting** | Sort by Date or Amount (ascending/descending) |
| **Search** | Search transactions by description |
| **Role-Based UI** | Toggle between Viewer (read-only) and Admin (add/edit/delete) |
| **Insights Section** | Highest spending category, monthly expense change, savings rate |
| **State Management** | Centralized state for transactions, filters, sorting, and role |
| **Responsive Design** | Works on desktop, tablet, and mobile devices |

### ✨ Additional Enhancements

- 🎨 Modern gradient background with smooth card hover effects
- 📱 Mobile-friendly responsive layout
- 🔄 Real-time chart updates when transactions change
- 🗑️ Delete confirmation (Admin role)
- ✏️ Inline edit functionality via modal form
- 📊 Empty state handling for no transactions
- 🎭 Role-based action buttons visibility
